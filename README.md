# UAST Ihugh University Website

The official website for the **University of Agriculture, Science & Technology (UAST), Ihugh** — a specialized university in Benue State, Nigeria focused on agricultural education, food security, and technological innovation.

**Live Site:** [https://soldierofgod-8.github.io/Uni-Website](https://soldierofgod-8.github.io/Uni-Website)

## Tech Stack

- **Framework:** Next.js 16 (App Router, static export)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **3D Graphics:** Three.js + React Three Fiber + Drei
- **Deployment:** GitHub Pages

## Getting Started

```bash
# Install dependencies
npm install

# Run the development server
npm run dev

# Build for production
npm run build
```

## Project Structure

```
uast-university-website/
  src/
    app/              # Next.js App Router pages
    components/       # Reusable components (3D, layout, UI, home sections)
    lib/              # Constants and site data
  public/             # Static assets (logos, icons)
```

## Features

- **Interactive Programme Explorer** — Browse 20 pioneer programmes across 4 schools with search and filtering
- **3D Animated Logo** — Rotating globe with university branding and energy ring animations
- **Glassmorphism UI** — Dark theme with translucent cards and green accents
- **Animated Transitions** — Scroll-reveal and page entrance animations via Framer Motion
- **Responsive Navigation** — Fixed navbar with dropdown menus for About, Academics, and Campus sections
- **Contact Form** — Client-side validated form with transparency documents section
- **Partnerships & Endowment** — Detailed pages for strategic collaboration and donation opportunities
